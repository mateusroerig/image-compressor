# image-compressor

Image compressor application built with [Micronaut](https://micronaut.io/).

## Requirements
- Micronaut 3.7.1
- JDK 11

## Run application
```shell
./gradlew run
```

To run in a specific port, change file `src/main/resources/application.yml`:
```yaml
# ...
server:
  port: 8081 # change to desired port, default 8080
```
